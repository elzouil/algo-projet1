# Projet AlgoBox - TP Collaboratif
Projet réalisé par Ramoni Elsa et Tardy Blandine.

## Structure du projet
- Le dossier `sources` contient les fichiers sources `.alg`.
- Le fichier principal est `sources/main.alg`.

## Comment ouvrir et exécuter le projet

1. **Prérequis** : Avoir le logiciel [AlgoBox](https://www.xm1math.net/algobox/) installé.
2. **Ouverture** :
   - Lancer AlgoBox.
   - Cliquer sur "Ouvrir".
   - Naviguer vers le dossier `sources` de ce dépôt.
   - Sélectionner le fichier `main.alg`.
3. **Exécution** :
   - Une fois le code chargé, cliquer sur le bouton "TESTER ALGORITHME".
   - Cliquer sur "LANCER ALGORITHME" dans la fenêtre qui s'ouvre.
